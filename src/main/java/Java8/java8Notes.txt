----------------function()--apply()-andTHen()--compose()-------
-he single abstract method present inside Function<T,R> is:R apply(T t);

-Explanation
 T = input type
 R = return type
 apply() takes one argument and returns one value.

 These are default/static methods, not abstract methods:

 andThen()  //default
 compose()  //default
 identity()//static method
-In Java 8, apply(), andThen(), compose(), and identity() methods are mainly used with
the Function<T,R> functional interface.

-apply() is used to perform an operation on input data and return a result.
-andThen() executes one function first and then passes the result to another function
-compose() works in reverse order.
-identity() returns the same value that is passed as input without changing it.

Syntax =>
import java.util.function.Function;

public class FunctionDemo {
    public static void main(String[] args) {

        Function<Integer, Integer> square = x -> x * x;
        Function<Integer, Integer> doubleValue = x -> x * 2;

        // apply()
        System.out.println(square.apply(5)); // 25

        // andThen()
        System.out.println(square.andThen(doubleValue).apply(5));
        // square first: 25, then double: 50

        // compose()
        System.out.println(square.compose(doubleValue).apply(5));
        // double first: 10, then square: 100

        // identity()
        Function<Integer, Integer> identityFun = Function.identity();
        System.out.println(identityFun.apply(5)); // 5
    }
}
